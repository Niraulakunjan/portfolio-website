Django Setup Documentation

1. Create Virtual Environment

python -m venv env

2. Activate Virtual Environment

# Windows 
env\Scripts\activate

3. Install Django

pip install django

4. Create Project

django-admin startproject myproject

5. Run Development Server

cd myproject
python manage.py runserver

6. Create an App

python manage.py startapp myapp

7. Register App in Settings

# In myproject/settings.py
INSTALLED_APPS = [
    ...
    'myapp',
]

8. Run Migrations

python manage.py makemigrations
python manage.py migrate

9. Create a View

# In myapp/views.py
from django.http import HttpResponse

def home(request):
    return HttpResponse("Hello, Django!")

10. Add App URL

# In myapp/urls.py
from django.urls import path
from .views import home

urlpatterns = [
    path('', home, name='home'),
]

11. Include App URLs in Project

# In myproject/urls.py
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('myapp.urls')),
]

12. Create and Use Templates

# Folder structure:
myapp/
└── templates/
    └── myapp/
        └── home.html

# In myapp/views.py
from django.shortcuts import render

def home(request):
    return render(request, 'myapp/home.html')

13. Settings for Templates

# In myproject/settings.py, ensure this line is in TEMPLATES:
'DIRS': [BASE_DIR / "templates"],

14. Using Static Files

# Folder structure:
myapp/
└── static/
    └── myapp/
        └── style.css

# In home.html (template)
{% raw %}
{% load static %}
<link rel="stylesheet" href="{% static 'myapp/style.css' %}">
{% endraw %}

15. Settings for Static Files

# In myproject/settings.py
STATIC_URL = 'static/'

# Add this if you use global static folders:
STATICFILES_DIRS = [BASE_DIR / "static"]

17. Static Folder Structure

# Recommended structure inside your Django app (e.g., 'myapp'):

myapp/
├── static/
│   └── myapp/
│       ├── style.css
│       ├── script.js
│       └── logo.png

# You can reference files like this in templates:
{% raw %}

{% endraw %}

16. Add Static Files in HTML Template

<!-- At the top of your HTML template -->
{% raw %}
{% load static %}
{% endraw %}

<!-- Use static files like this -->
<link rel="stylesheet" href="{% static 'myapp/style.css' %}">
<script src="{% static 'myapp/script.js' %}"></script>
<img src="{% static 'myapp/logo.png' %}" alt="Logo">
